Features of Code
- CSV Upload: Users can upload any CSV file using the drag-and-drop or file selection component.
- Dynamic Table Display: Shows a preview of the uploaded data in a table format.
- Graphs Adapt to Data: If the dataset has at least 2 columns, a bar chart is generated. With at least 3 columns, a line chart is added. If 4 or more columns exist, a pie chart is displayed.
- Flexibility: The app works for any CSV file, whether it's inventory, lottery numbers, or another dataset.
